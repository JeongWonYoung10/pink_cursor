# Overview

Blush Command System is a blush-first design system for AI coding workspaces, product landing pages, and slide decks.

## Definition

A soft command center for creative code: warm surfaces, sharp workflows, and human-controlled AI.

## v1 Scope

- Foundations: color, typography, radius, shadow, spacing, motion
- Primitives: Button, Card, Badge, Input, Progress, Toast
- Product components: AgentBubble, CommandRow, DiffLine, FileTreeItem, EditorTab, StatusPill
- Core patterns: Agent Panel, Command Palette, Diff Review, Editor Shell, Journey Demo
- Examples: Landing, Product Showcase, Workflow Demo, Slide Preview

## Positioning

Not a cute pink coding editor.  
A soft, trustworthy AI coding workspace system with precise human control.
