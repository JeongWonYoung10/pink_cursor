# Product Patterns

## Agent Panel

The cockpit where users steer AI work. It includes status, context, conversation, risk/memory, control dock, and input.

## Command Palette

The launcher for actions, files, checks, and agent workflows. It combines search, command rows, groups, and shortcuts.

## Diff / Review

The trust layer where AI work becomes human-approved work. It includes file navigator, diff canvas, checklist, decision dock, and recovery.

## Editor Shell

The workbench where code, context, and agent actions meet. It includes file tree, tabs, code canvas, terminal, status bar, and agent rail.

## Journey Demo

The end-to-end workflow: Prompt → Plan → Edit → Review → Merge.
