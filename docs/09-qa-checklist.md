# QA Checklist

- [ ] Is there only one dominant primary action per screen?
- [ ] Is hot pink used for role/state, not decoration?
- [ ] Does the agent explain what it is doing?
- [ ] Can the user approve, edit scope, pause, revert, preview, or reject?
- [ ] Are file names, diff counts, checks, and status values visible?
- [ ] Does error copy explain the cause and provide a recovery action?
- [ ] Is the code readable?
- [ ] Does the screen feel like a trustworthy AI coding tool, not just a cute pink UI?
