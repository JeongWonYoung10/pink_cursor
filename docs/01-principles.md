# Principles

## 1. Blush is the environment, not the accent
Use blush as atmosphere and soft surface. Use hot pink for action, selection, and state.

## 2. Keep the workflow sharp
Every screen should clarify the current step and the next action.

## 3. The user controls the agent
AI can propose and execute. The user approves, scopes, pauses, reverts, and merges.

## 4. Technical information stays precise
File names, diff counts, checks, branches, status labels, and code must remain clear.

## 5. Glow means state
Glow is for active, running, selected, and success states. Avoid decorative glow overload.
