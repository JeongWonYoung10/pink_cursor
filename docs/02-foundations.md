# Foundations

## Color

- Primary Action: `#FF4F9A`
- Action Hover: `#FF5DA2`
- Soft Surface: `#FFD1DC`
- Panel Surface: `#F8C8DC`
- Mist Depth: `#EFCFE3`
- Warm Accent: `#FF9A9E`
- Base: `#FFF7FA`, `#FFFFFF`
- Text Strong: `#3A2030`
- Text Muted: `#7F5268`, `#8A6074`, `#9B6C82`

## Type

The v1 artifact uses the same font behavior as the canvas prototype: system UI fonts, not a bundled or externally loaded webfont.

- UI / Web / Slide: `ui-sans-serif, system-ui, sans-serif`
- Code / Diff / Terminal: `ui-monospace, SFMono-Regular, Menlo, Monaco, Consolas, monospace`

Display creates emotion. Body explains. Meta builds trust. Code preserves technical precision.

## Surface Levels

1. Background: `#FFF7FA` + soft gradients
2. Glass shell: white 45–60%
3. Card/panel: white 55–75%
4. Inner block: blush 25–45%
5. Active surface: `#FF4F9A` or gradient

## Radius

Large radius is part of the product language. Use less radius for dense technical rows and more radius for emotional containers.

## Shadow

Use shadow for depth and state. Technical rows should stay flat.
